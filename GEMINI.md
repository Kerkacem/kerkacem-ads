# [PLATFORM_NAME] - Project Requirements

## Core Philosophy
AI-powered advertising intelligence for Meta Ads, e-commerce, and COD sellers in MENA.

## Key Features
- **Dashboard:** Unified KPI tracking, multi-account support, AI-driven performance insights.
- **Hub Agent IA:** Centralized chat interface for campaign analysis, budget optimization, ad copy/creative generation.
- **Spy Tool:** Competitor ad intelligence, search/filter, creative analysis.
- **WinFinder:** Winning product discovery, demand analysis, margin calculation.
- **Product Database:** Product inventory, profit analysis, AI-powered sales intelligence.
- **Performance Reporting:** Comprehensive reports with AI executive summaries.

## Technical Constraints
- **Localization:** RTL (Arabic), French, English.
- **Currency:** DZD (Primary), configurable per workspace.
- **Theme:** Dark mode (#0a0a0a), orange accents (#f97316), Inter font.
- **Infrastructure:** Node.js (Fastify) Backend, Next.js Frontend, PostgreSQL + Redis.

## Roadmap Phases
1. **Infrastructure & Auth:** (In Progress) Workspace/User/Subscription/RBAC.
2. **Dashboard & API:** Meta Graph API integration, KPI tracking.
3. **AI Command Center:** Hub Agent IA engine + chat interface.
4. **Spy & WinFinder:** Product/Ad search and analysis logic.
5. **Reporting & Analytics:** Automated report generation engine.
